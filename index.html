Hexaletter Rules

Aw yeah man! This is a super radical hexagonal
based word game! You gotta try it out!
Click hex tiles one connecting tile at a time to create a word that match a theme.
Click the 'Submit Word' button if you think you have a valid word.
If it's a valid word in the list that match the theme, it'll turn...green.
If not, it'll just remain white.
Try to clear the board Good luck!
